# Automated Weather Pipeline

This project builds an automated data pipeline that fetches weather forecasts from the Open-Meteo API, stores the data in a SQLite database, generates a bilingual weather poem (English + Bengali) using Groq LLM, and publishes the result on a GitHub Pages website.

Locations used:
- Kushtia (Bangladesh)
- Dhaka (Bangladesh)
- Aalborg (Denmark)

The pipeline runs automatically every day using GitHub Actions.

Live website:
